Write short note on a web browser.

A web browser helps you surf the internet. It retrieves information from other parts of the web and displays it on your desktop or mobile devices. A web browser transfers the data using the HTTP (Hypertext Transfer Protocol) . HTTP defines how text, images and videos are transmitted on the web. People can use internet from anywhere in the world by using a web browser.
